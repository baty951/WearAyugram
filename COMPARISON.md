# Сравнение функций: WearAyugram vs AyuGram

Сравнение WearAyugram (самостоятельный клиент для WearOS) с AyuGram для Android
(форк на базе exteraGram) и AyuGram Desktop (форк на базе Telegram Desktop / tdesktop).

Актуально на 2026-06-11.

## Ayugram-фичи

| Функция | WearAyugram (часы) | AyuGram (Android) | AyuGram Desktop |
|---|---|---|---|
| Ghost Mode | ✅ не шлёт ViewMessages/OpenChat/online | ✅ гибкий, по компонентам | ✅ гибкий, быстрый тумблер |
| Anti-revoke / Anti-Recall | ✅ Room DB, серый пузырь 🚫 | ✅ постоянное хранилище | ✅ |
| Уведомления/метки об удалении | ✅ нотификация | ✅ метки в чате | ✅ метки в чате |
| Local Premium | ✅ | ✅ | ✅ |
| История редактирований | ❌ (только метка «изменено») | ✅ | ✅ |
| Кастомные метки deleted/edited | ❌ | ✅ | ✅ |
| Скрытие рекламы | ❌ (и так не запрашивается) | ✅ | ✅ по умолчанию |
| Снятие запретов копирования/скачивания | ❌ | ✅ | ✅ |
| Скрытие сообщений от заблокированных | ❌ | ⚠️ частично | ✅ полное |
| AyuSync | ❌ | ✅ | ✅ |
| Streamer Mode | ❌ | ❌ | ✅ |
| Message Shot (скриншот сообщения) | ❌ | ❌ | ✅ |

## Выводы

- Все четыре ключевые Ayugram-фичи (Ghost Mode, Anti-revoke, уведомления об
  удалении, Local Premium) на часах есть и работают на уровне старших клиентов.
- Самый заметный разрыв — сервисные фичи, которые есть у обоих старших клиентов:
  история редактирований, кастомные метки, AyuSync.
- Звонки и истории на часах осознанно не реализованы — на этом форм-факторе
  они почти не имеют смысла.
- Кандидаты на следующие фазы: история редактирований (инфраструктура
  `UpdateMessageEdited` + Room уже есть) и реакции (полезны при чтении с часов).
