## Действия над сообщениями (дёшево, инфраструктура готова)

Меню действий (долгое нажатие) уже умеет реакции/ответ/пересылку — туда же:

- **Удаление** сообщения (`DeleteMessages`, для себя / для всех)
- **Редактирование** своего сообщения (`EditMessageText`) — RemoteInput уже используется
- **Копировать текст** в буфер
- **Закрепить / открепить** + экран закреплённых (`PinChatMessage`, `GetChatPinnedMessage`)
- **Ответить с цитатой** выделенного фрагмента (TextQuote уже мапится)

## Управление списком чатов (дёшево)

- **Mute/unmute прямо с часов** — mute-статус уже читается, нужен переключатель (`SetChatNotificationSettings`)
- **Закрепить/открепить чат**, **архив** (отдельный список, как папки), **выйти/удалить чат**
- **Отметить прочитанным/непрочитанным**, кнопка **«прочитать все»**

## Непокрытые типы контента

- ~~Опросы~~ — сделано (голосование, множественный выбор)
- ~~Геолокация, контакт, сервисные сообщения~~ — сделано (текстовые представления)

- **Альбомы** (сгруппированные фото) — сейчас каждое отдельным пузырём
- **Спойлер-медиа** (`hasSpoiler`) — блюр до тапа

## Профили и инфо (среднее)

- **Экран профиля** пользователя/группы: bio, username, телефон, участники, общие чаты
- **Настоящие аватары** вместо буквы+цвета (фото чатов уже качаются — переиспользовать в ChatListScreen)

## Оставшиеся фичи Ayugram

- **Кастомные метки** deleted/edited (настраиваемый текст/значок)
- **Регекс/ключевые слова** — фильтрация сообщений
- **Детали сообщения** (автор пака стикеров/эмодзи)
- AyuSync, Streamer Mode — мимо форм-фактора

## Переводчик (среднее, TDLib умеет)

- **Перевод сообщения** по тапу (`TranslateText`/`TranslateMessageText`)

## Голос / распознавание (дёшево — поле уже в модели)

- **Транскрипция голосовых** (`RecognizeSpeech`, `speechRecognitionResult` уже есть в VoiceNote/VideoNote)

## Анимация и эмодзи (крупное)

- **Анимированные стикеры** (TGS/WEBM вместо статичного кадра) — нужен Lottie/видео-пайплайн
- **Кастомные эмодзи-реакции** (премиум-стикеры в реакциях)

## Wear-специфика (полировка)

- **Quick-reply из уведомления** (RemoteInput-экшен прямо в нотификации)
- **Complication** для циферблата (счётчик непрочитанных — Tile уже есть)
- ~~Несколько аккаунтов + переключение, выход~~ — сделано (3 слота)
- **Настройки**: размер шрифта, тема, язык интерфейса (сейчас RU-строки захардкожены)
- **AmbientMode** — в README числится «не поддерживается», можно вернуть

## Рекомендации (польза / цена)

1. **Удаление + редактирование своих сообщений** — очевидный пробел, инфраструктура готова
2. **Mute/unmute с часов** — завершает уже сделанную mute-логику, один вызов API
3. **Транскрипция голосовых** — поле в модели уже есть, на часах ценнее прослушивания
4. **Настоящие аватары** — заметный визуальный апгрейд, фото уже скачиваются
